Kebijakan Privasi

Terakhir diperbarui: 3 Juli 2025

Aplikasi TapCloud menghormati privasi pengguna dan berkomitmen untuk menjaga data pribadi Anda dengan aman. Kami hanya mengumpulkan informasi yang diperlukan untuk menjalankan game dan memberikan pengalaman terbaik kepada pengguna LINE.

Data yang Dikumpulkan

  • Nama tampilan (display name) dari akun LINE Anda
  • LINE user ID (ID pengguna unik)
  • Foto profil (jika tersedia)

Penggunaan Data

Data digunakan untuk:

  • Menyimpan skor dan progres game Anda
  • Menampilkan leaderboard dan profil pengguna
  • Memberikan pengalaman personal dalam game

Penyimpanan & Keamanan

Data Anda disimpan di server kami secara aman dan tidak dibagikan ke pihak ketiga tanpa izin.

Hak Anda

Anda berhak meminta penghapusan data Anda kapan saja dengan menghubungi kami melalui email di tapcloud.support@gmail.com.

Dengan menggunakan TapCloud, Anda menyetujui kebijakan privasi ini.
